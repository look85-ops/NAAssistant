# Сервис подбора вакансий

Локальный Python-сервис для парсинга hh.ru по критериям Натальи. Ищет вакансии instructional design, L&D, методист, edtech и смежные — с фильтрацией нерелевантного.

**Файлы:**
- `job-search.py` — парсер и ранжирование
- `serve.py` — локальный HTTP-сервер (порт 8080)
- `vacancies.html` — HTML-отчёт с результатами (генерируется)

**Критерии:**
- Ключевые слова: instructional design, L&D, методист, edtech, training manager, learning experience
- Исключения: продажи, дети/школы/репетиторы, 1С/бухгалтерия, английский
- Бустинг: корпоративное обучение, adult, L&D, HR, edtech
- Зарплата от: 200 000 ₽
- Регионы: РФ (113), РБ (16)

**Как запустить:**
```powershell
python scripts\serve.py
```
Открыть http://localhost:8080/vacancies.html  
Кнопка «Обновить» в интерфейсе перезапускает парсер.

**Стек:** Python (urllib, re), HTML/CSS/JS (статика), без внешних зависимостей.
