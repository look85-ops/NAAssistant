# Портфолио L&D-специалиста

Сайт-визитка Натальи Марценюк. Демонстрация кейсов и компетенций в области L&D, методологии и педагогического дизайна.

**URL:** https://look85-ops.github.io/NAAssistant/portfolio/ (GH Pages)  
**Деплой:** GitHub Pages (https://look85-ops.github.io/NAAssistant/portfolio/)

**Стек:** HTML, CSS, JS. Статика (без серверной части).

**Структура:**
- `index.html` — главная (профиль, кейсы, контакты)
- `message-accidental_1.html` — отдельный проект «Послание случайно»
- `draft/` — черновик редизайна
- `materials/` — примеры работ (pdf)

**Как обновлять:** правишь файлы в этой папке, коммитишь в `main` — GH Pages деплоит автоматически.
