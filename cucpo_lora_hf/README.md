# 🍳 Cucpo LORA + Методист

LoRA-адаптер на базе **Qwen/Qwen2.5-3B-Instruct** с навыками методиста-разработчика курсов.

## Возможности

- **Методическая экспертиза**: разработка курсов, педагогический дизайн, оценка эффективности
- **Инструменты**: веб-поиск (DuckDuckGo), загрузка страниц
- **Персонализация**: система сама использует скиллсет методиста

## Использование

1. Напишите запрос в чате (например: "Как спроектировать модуль курса?")
2. Используйте поиск для актуальной информации
3. Модель ответит с позиции методиста-эксперта

## Деплой

Этот Space загружает LoRA из внешнего пути. При запуске:
- Скачивается базовая модель Qwen2.5-3B-Instruct
- Применяется адаптер из `cucpo_lora_output`

## Требования

- GPU (рекомендуется для inference)
- ~8GB VRAM
