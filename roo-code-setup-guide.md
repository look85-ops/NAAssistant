# Настройка Roo Code + BotHub: пошаговая инструкция

## Шаг 1. Установи VS Code

Скачай и установи Visual Studio Code: https://code.visualstudio.com/

Если VS Code уже стоит — переходи к шагу 2.

## Шаг 2. Установи расширение Roo Code

1. Открой VS Code
2. Нажми на иконку Extensions (квадратики) в боковой панели слева — или `Ctrl+Shift+X`
3. В поиске набери **Roo Code**
4. Нажми **Install** на расширении от `RooCode` (иконка — голубой кенгуру)
5. После установки в боковой панели появится иконка Roo Code — кликни на неё

> Откроется чат-панель — это и есть твой интерфейс для работы с AI.

## Шаг 3. Подключи BotHub как провайдер

1. В панели Roo Code нажми на **иконку шестерёнки** (Settings) в правом верхнем углу чата
2. Найди раздел **API Provider** — выбери **OpenAI Compatible**
3. Заполни поля:

| Поле | Значение |
|------|---------|
| **Base URL** | `https://bothub.chat/api/v2/openai/v1` |
| **API Key** | Твой ключ BotHub (`sk-...`) |
| **Model ID** | `glm-5.1` |

4. Нажми **Save** / подтверди настройки


## Шаг 5. Проверь, что всё работает

1. В чате Roo Code напиши: `Привет! Скажи, какая ты модель?`
2. Если видишь ответ — всё настроено
3. Если ошибка `401 Unauthorized` — проверь API-ключ
4. Если ошибка `404 Not Found` — проверь Base URL (без лишних пробелов и слешей в конце)


## Что попробовать первым делом

1. Открой любой проект (или создай пустую папку)
2. Напиши в чат: `Создай простое веб-приложение — калькулятор на HTML/CSS/JS`
3. Roo Code создаст файлы, покажет diff, попросит подтверждение
4. Подтверди — файлы появятся в проекте
5. Открой `index.html` в браузере — готово!
