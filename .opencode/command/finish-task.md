# /finish-task

Закрыть текущую задачу и зафиксировать результат.

## Процесс

1. **Проверить критерии готовности:**
   - Пробежаться по всем чекбоксам в описании фичи
   - Если что-то не сделано → сообщить пользователю

2. **Обновить статус** фичи на "Готово"

3. **Зафиксировать результат:**
   - Что было сделано (bullet points)
   - Сколько времени заняло (если известно)
   - Были ли проблемы/отклонения

4. **Закоммитить изменения:**
   - Показать `git diff` пользователю
   - Спросить "Коммичу?"
   - Если да → `git add` + `git commit` с описанием
   - Формат коммита: `[F-NNN] Название фичи`

5. **Обновить BACKLOG.md:**
   - Статус → "Готово"
   - Добавить дату закрытия

6. **Предложить следующую задачу:**
   - Показать следующую фичу в приоритете
   - Спросить "Берём F-NNN?"

## Когда вызывать

Когда пользователь говорит:
- "Закрой задачу"
- "Готово, коммить"
- "Закончили, коммитим"
- Фича выполнена

## Формат коммита

```
[F-NNN] Название фичи

- Что сделано 1
- Что сделано 2
```
